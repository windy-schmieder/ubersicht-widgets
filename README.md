# Übersicht Widgets

A collection of personal [Übersicht](https://tracesof.net/uebersicht/) widgets for macOS.

> **Note:** I am not the original author of these widgets. They are edited/modified versions of widgets created by others. Credit goes to their respective original creators.

---

## Widgets

### do-i-need-a-jacket.jsx
Displays current weather conditions and a simple recommendation on whether you need a jacket. Uses [wttr.in](https://wttr.in) — no API key required. Edit the `CITY` variable at the top of the file to set your location.

### simple-calendar.jsx
A minimal calendar widget that displays the current month with today's date highlighted.

### spotify-now-playing.jsx
Shows the currently playing Spotify track, including song title, artist, album, and album artwork.

---

## Installation

1. Install [Übersicht](https://tracesof.net/uebersicht/)
2. Clone this repo into your Übersicht widgets folder:
   ```
   git clone https://github.com/windy-schmieder/ubersicht-widgets ~/Library/Application\ Support/Übersicht/widgets/ubersicht-widgets
   ```
3. Übersicht will automatically pick up the widgets.
